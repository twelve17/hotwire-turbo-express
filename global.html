TurboMultipleStreamActionFunction(view, variables, stream, onlyFormatopt)

Function for sending multiple view templates, each getting wrapped in its own tag.
Parameters:
Name Type Attributes Default Description
view string Path to the view view to render inside the tag.
variables object Variables to pass on to the view, in the same format as is done for req.render.
stream object Attributes to set in the tag. Note that "action" here will be ignored, as it will be set by the function itself.
onlyFormat boolean <optional>
 false If true, will wrap the response in a "res.format" object, so that the route will _only_ respond to the Turbo mime type, or return a HTTP 406 (Not Acceptable) response.
